ATM Simulation System

This project is a simulation of a basic ATM system developed using full stack technologies. It enables users to securely log in and perform essential banking operations such as: Login and logout functionality with session-based access. Withdrawal and deposit operations with proper validations. Display of the current account balance after each transaction. Basic transaction handling to ensure accurate balance updates. Secure input handling and error checking for a reliable experience. This project demonstrates practical implementation of authentication, transaction logic, and user interface design for a simple banking application.

Project 3

mu-react-store

mu-react-store is a full stack e-commerce web application built using React.js for the frontend and Node.js with SQL for the backend. The application allows users to register, log in, and browse a list of products. Each product displays an image, name, description, price, and an "Add to Cart" option. User authentication is handled securely with backend validation. The project demonstrates integration between frontend and backend using API calls and database connectivity.
